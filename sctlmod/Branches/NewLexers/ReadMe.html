ScintillaNET

ScintillaNET is a powerful text editing control for Windows® Forms applications and a managed wrapper around the versatile Scintilla component. Created with the developer in mind, the ScintillaNET API makes it simple to add advanced text editing and syntax highlighting to your application or IDE. In addition ScintillaNET adds features not found in other Scintilla wrappers such as Visual Studio® style code snippets, integrated find and replace dialogs, true regular expression searches, multiple key-command bindings, and back/forward document navigation.

There are a few important things to know about using ScintillaNET:

  • ScintillaNET is a wrapper around a native C++ library called SciLexer.dll.
  • Starting with ScintillaNET v3.0, the SciLexer.dll (for 32 and 64-bit platforms) comes embedded in the ScintillaNET.dll. During execution the embedded SciLexer.dll is written to the user's temp folder and then loaded from disk.
  • This requires that your application have read/write access to the user's temp folder—which most applications do.
  • You can override this behavior if your application doesn't have permissions to the user's temp folder or you would simply prefer to load the SciLexer.dll from another location by calling Scintilla.SetModulePath prior to any Scintilla controls being created.

The ScintillaNET project is hosted on CodePlex at the following URL: http://scintillanet.codeplex.com. The most up-to-the-minute source code, binaries, and documentation can be found at that location.
